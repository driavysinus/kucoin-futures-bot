# Интеграция ценовых алертов — KuCoin Futures Bot

## Что добавлено

Система ценовых алертов + полное управление ордерами через консоль. При достижении заданной цены по WebSocket автоматически открывается рыночная позиция с полной существующей логикой сопровождения (тейки 50/25/25%, перенос стопа, безубыток).

## Разбор команды `add`

```
add XBTUSDTM 70000 buy 100 68000 2 10
 │      │       │    │   │    │    │  │
 │      │       │    │   │    │    │  └─ LEV    = 10x плечо
 │      │       │    │   │    │    └─── TRIG%  = 2% — шаг между тейками
 │      │       │    │   │    └─────── SL     = 68000 — стоп-лосс
 │      │       │    │   └──────────── USDT   = 100 — размер позиции
 │      │       │    └──────────────── SIDE   = buy → LONG
 │      │       └───────────────────── PRICE  = 70000 — цена срабатывания
 │      └───────────────────────────── SYMBOL = торговая пара
 └──────────────────────────────────── команда
```

**Что произойдёт:**
1. Алерт сохраняется в памяти
2. Бот подписывается на тикер XBTUSDTM по WebSocket
3. Когда цена опустится до 70000 → маркет-ордер LONG на 100 USDT
4. Стоп-лосс на 68000
5. Тейк 1: 50% позиции при +2% от входа → стоп в безубыток
6. Тейк 2: 50% остатка при ещё +2% → стоп на цену тейка 1
7. Тейк 3: весь остаток при ещё +2% → позиция закрыта
8. Telegram: уведомления на каждом шаге

## Все консольные команды

### Алерты

| Команда | Описание |
|---------|----------|
| `add SYMBOL PRICE SIDE USDT [SL] [TRIG%] [LEV]` | Добавить ценовой алерт |
| `list` | Показать активные алерты |
| `remove ID` | Удалить алерт по номеру |
| `clear [SYMBOL]` | Удалить все алерты (или по символу) |

### Ордера и позиции

| Команда | Описание |
|---------|----------|
| `orders [SYMBOL]` | Показать активные ордера на бирже |
| `positions` | Показать открытые позиции |
| `cancel ORDER_ID` | Отменить конкретный ордер |
| `cancelall SYMBOL` | Отменить все ордера по символу |
| `close SYMBOL [PCT]` | Порез позиции (по умолч. 50%) |
| `price SYMBOL` | Текущая цена (WS кэш или REST) |
| `help` | Справка |

### Примеры

```bash
# Алерты
add XBTUSDTM 70000 buy 100 68000 2 10
add ETHUSDTM 4000 sell 50 4200 1.5 5
add BTC 95000 buy 20 93000 1 3        # BTC → BTCUSDTM автоматически
list
remove 2

# Ордера
orders                                  # все ордера
orders XBTUSDTM                         # только по BTC
cancel 5f3b7c8d9e1a2b3c4d5e6f7a        # отменить ордер по ID
cancelall XBTUSDTM                      # отменить все по символу

# Позиции
positions
close XBTUSDTM 50                       # закрыть 50% позиции
close ETHUSDTM                          # закрыть 50% (по умолч.)

# Цена
price BTC
price XBTUSDTM
```

## Поток данных

```
Консоль: add XBTUSDTM 70000 buy 100 68000 2 10
                ↓
        AlertManager хранит алерт
                ↓
        FuturesMonitor подписывается на тикер
                ↓
WebSocket (public) → price_update → AlertManager.on_price_update()
                ↓
        Цена ≤ 70000 → СРАБОТАЛ
                ↓
        OrderManager.place_market_with_plan()
                ↓
        KuCoin REST: маркет-ордер + создание Plan
                ↓
WebSocket (private) → on_order_filled → _on_entry_filled
                ↓
        Тейк 1 (50%) + SL → безубыток
        Тейк 2 (25%) → стоп на тейк 1
        Тейк 3 (25%) → позиция закрыта 🏁
                ↓
        Telegram: уведомления на каждом шаге
```

## Логика срабатывания алертов

| SIDE | Триггер | Смысл |
|------|---------|-------|
| `buy` | цена **опускается** до trigger_price | Вход в лонг на поддержке |
| `sell` | цена **поднимается** до trigger_price | Вход в шорт на сопротивлении |

Для пробойных входов (buy при росте цены) используйте команду `/stop` в Telegram.
